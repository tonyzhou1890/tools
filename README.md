# tools

这是个人在线工具网站的介绍页。

|工具|简介|

|[字体查看器](https://dowhat.net/zh-CN/service/font)|这个工具可以查看 ttf、otf、woff 格式的字体，并且支持字体裁剪。页面上提供了一些开源字体。|
